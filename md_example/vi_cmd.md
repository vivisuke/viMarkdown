# vi コマンド・リファレンス (viMarkdown テスト用)

## 1. モードの切り替え
- `i` : カーソルの直前から挿入モードへ
- `a` : カーソルの直後から挿入モードへ (append)
- `o` : 現在の行の下に新しい行を作って挿入モードへ
- `ESC` : コマンドモードに戻る

## 2. カーソル移動 (基本)
| コマンド | 動作 |
| :--- | :--- |
| `h` | 左へ移動 |
| `j` | 下へ移動 |
| `k` | 上へ移動 |
| `l` | 右へ移動 |
| `0` | 行頭へ移動 |
| `$` | 行末へ移動 |
| `gg` | 文頭へ移動 |
| `G` | 文末へ移動 |

## 3. 編集コマンド
- `x` : カーソル上の1文字を削除
- `dd` : 現在の行を削除（切り取り）
- `yy` : 現在の行をコピー (yank)
- `p` : カーソルの下に貼り付け (paste)
- `u` : 直前の操作を取り消す (undo)
- `Ctrl + r` : 取り消しを取り消す (redo)

## 4. 検索・置換
- `/文字列` : 文字列を前方検索
- `n` : 次の一致へ移動
- `N` : 前の一致へ移動

## 5. 保存と終了
- `:w` : 保存
- `:q` : 終了
- `:wq` : 保存して終了
- `:q!` : 保存せずに強制終了
